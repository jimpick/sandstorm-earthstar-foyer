sandstorm-earthstar-foyer
=========================

An experiment to to see if I can package
[Earthstar Foyer](https://github.com/earthstar-project/earthstar-foyer)
to run on [Sandstorm](https://sandstorm.io/) and get it to sync between
two different Sandstorm servers.
